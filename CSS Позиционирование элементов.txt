
----------------------------------- Потоки документа -------------------------------------------

 Поток документа - Потоком документа в HTML называется порядок вывода элементов на страницу. 
                   В обычном виде все блоки выводятся в том порядке, в котором они записаны 
                   внутри HTML-документа.

 Нормальный поток документа - схема, при которой блочные и строчные HTML-элементы выводятся 
                              друг за другом в том порядке, в котором они размечены в файле.

 Плавающий поток. Создаётся с использованием CSS-свойства float.

 Поток с абсолютным позиционированием. Создаётся с использованием свойства position: absolute.


------------------------------ Относительное позиционирование ----------------------------------

 Относительное позиционирование - элемент будет позиционироваться относительно изначальной 
                                  позиции.
 position: relative; - после перемещения элемента пространство остается заполеннным им

 Пример использование position: relative и :hover:

 .card {                     .card:hover {
   position: relative;         top: -15px;
   top: 0px;                   left: -10px;  
   right: 0px;
  
   
---------------------------------- Абсолютное позиционирование -----------------------------------

 position: absolute; - отличительной чертой абсолютного позиционирования является то, что блок 
                       как бы "выдёргивается" из потока документа.\
 
 От чего позиционируется элемент при абсолютном позиционировании?
 Ответ:
 Относительно левого верхнего угла страницы.

 Если задавать 0 right и 0 left, то блок растянется на всю ширину.


------------------------- Относительное и абсолютное позиционирование ----------------------------   
   
 Если у родительского блока относительное или абсолютное позиционирование, то дочерний элемент 
 (с абсолютным позиционированием) будет располагаться относительно него, а не края страницы.
 
 Зачем применяется связка относительного и абсолютного позиционирования?
 Ответ: 
 Для позиционирования абсолютного элемента внутри относительного.


-------------------------------- Фиксированное позиционирование ----------------------------------
 
 position: fixed;

 Фиксированное позиционирование:
 1) Элемент также выдёргивается из нормального потока документа, а его место занимают 
    другие элементы;
 2) Будет перемещаться вместе с тем, как пользователь скролит страницу.
 3) Позиционируется относительно видимой области браузера.


-------------------------------------- Плавающие элементы ---------------------------------------

 Плавающий поток - когда текст обтекает элемент с раных сторон, например.
 С помощью этого можно создать блоки и внизу хуттер.

 float: left;

 Чтобы сменить поток на плавающий, используется CSS-правило float.
 1) left
 2) right

 Особенностью float в том, что при его использовании элемент автоматически становится блочным.
 При использовании float элемент удаляется из нормального потока документа.
 
 clear - очистить плавающий поток.
 1) left. Очистка плавающего потока слева.
 2) right. Очистка плавающего потока справа.
 3) both. Очистка плавающего потока слева и справа.

 Метод использования:
 1) Использование правила clear: both для элемента, который необходимо «очистить».
 2) Создать блочный элемент и указавать правило clear: both.
 3) Хз какая то непонятица.


-------------------------------------- Наложение элементов ---------------------------------------

 z-index - управлять наложением элементов.
 
 Важно, что свойство z-index работает только с элементами, у которых установлено 
 свойство position.
